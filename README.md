### 🧊 ポートフォリオをご覧いただくために

恐れ入りますが、「Data_analysis_using_the_Titanic_dataset.ipynb」ファイルをダウンロードいただき、Google Colaboratoryにアップロードをお願い致します。

上から順番に▶のボタンを押していただくと、全てのコードが動作します。
